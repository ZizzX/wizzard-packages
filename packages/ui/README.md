# @wizzard-packages/ui

![license](https://img.shields.io/npm/l/@wizzard-packages/ui)

Docs UI site for Wizzard Stepper. This package powers the documentation experience and is not meant for direct runtime usage.

## Local development

```bash
pnpm -C packages/ui dev
```

## Build

```bash
pnpm -C packages/ui build
```
